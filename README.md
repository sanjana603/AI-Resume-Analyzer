# 📄 AI Resume Analyzer

An AI-powered Resume Screening Application built using Python and Streamlit.

##  Features

- ATS Resume Match Score
- Resume vs Job Description Analysis
- Keyword Extraction
- Missing Skills Detection
- Interactive Dashboard
- AI-based Resume Suggestions

##  Technologies Used

- Python
- Streamlit
- Scikit-learn
- NLTK
- Matplotlib
- PyPDF2

##  Project Structure

```bash
AI-Resume-Analyzer/
│
├── app.py
├── requirements.txt
├── README.md
│
├── assets/
│   └── style.css
│
└── .streamlit/
    └── config.toml
```

## ▶ Run Project

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

##  Output

- ATS Match Score
- Resume Feedback
- Missing Skills Analysis
- Dashboard Visualization
